# Simple Java Console Calculator

This project serves as a practical implementation of basic Java programming concepts, including user input processing and conditional logic.

## Features
- Basic Arithmetic: Supports Addition, Subtraction, Multiplication, and Division.
- Input Validation: Includes checks to prevent errors such as division by zero.
- Precision: Outputs results formatted to two decimal places for clarity.

## Code Structure
- Scanner Class: Utilized for capturing user input from the console.
- Switch Expressions: Implemented to efficiently handle operator selection.
- Error Handling: Logic included to manage mathematical impossibilities and invalid operators.

## Prerequisites
- Java Development Kit (JDK) 8 or higher.
- A terminal or an Integrated Development Environment (IDE) like IntelliJ IDEA, Eclipse, or VS Code.

#  Java Swing Notepad

A simple desktop text editor built using **Java Swing**, inspired by Windows Notepad.

##  Features
- Create new text files
- Open existing `.txt` files
- Save text to file
- Scrollable text area
- Keyboard shortcuts (Ctrl+N, Ctrl+O, Ctrl+S, Ctrl+Q)

##  Tech Stack
- Java
- Swing (GUI)
- AWT
- File I/O

##  How to Run
```bash
javac Notepad.java
